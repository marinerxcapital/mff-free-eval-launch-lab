# MFF Free Eval Launch Lab

Unofficial proof-of-work concept for a premium AI-built campaign website.

## Pages

- `/` — Free Eval Landing Page
- `/studio` — AI Campaign Studio
- `/case-study` — Case Study

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Mock AI campaign generator
- Zod validation
- API routes ready for LLM integration

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Scripts

- `npm run dev` — Start dev server
- `npm run build` — Production build
- `npm run start` — Start production server
- `npm run lint` — Lint
- `npm run typecheck` — TypeScript check
- `npm run health` — typecheck + build
